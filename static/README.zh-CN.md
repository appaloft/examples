# Static

用于测试 Appaloft **静态站上传** 的最小网站。

无需构建、无后端、无依赖，只有 HTML / CSS / 少量 JavaScript。

## 使用方式

1. 下载本目录（或整个 examples 仓库 ZIP 后打开 `static/`）。
2. 将文件夹或 ZIP 上传到 Appaloft 静态站部署入口。
3. 打开部署 URL，确认看到 Appaloft Hello 页面。
