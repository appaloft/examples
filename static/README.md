# Static

Minimal static website for testing Appaloft **static site upload**.

No build step, no server, no dependencies — HTML, CSS, and a little JavaScript.

## Use

1. Download this folder (or the whole repo ZIP and open `static/`).
2. Upload the folder or ZIP to Appaloft static site deploy.
3. Open the published URL and confirm the “Appaloft Hello” page.
